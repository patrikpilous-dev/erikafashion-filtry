# Erika Fashion · Filtry — prodejnost podle atributů

**Live:** https://patrikpilous-dev.github.io/erikafashion-filtry/

Analyzuje 16 user-facing filtrů (Barva, Materiál, Velikost, Rukáv, Střih, Délka šatů, …) napříč 23 591 produkty.

## Sekce
1. Přehled všech 16 filtrů (souhrn)
2. Per filtr — breakdown hodnot (dropdown)
3. Heatmapa hodnota × kategorie (multi-select kategorie)
4. Kombinace filtrů (TOP 50 párů, lift)
5. Dead filter values (mrtvé hodnoty se skladem)

## Klíčová metrika
**Prům. obrat / produkt** = obrat / počet produktů s touto hodnotou. Ukazuje reálnou sílu hodnoty filtru (kompenzuje case kdy má hodnota X produktů, ale jen 1/3 prodává).
